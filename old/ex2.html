Что такое CMS

CMS - "система управления конетонтом" (движок) - написанная PHP-программистами для сайта с помощью которой вы сможете управлять сайтом (добавлять контент, менять пункты меню и т.п.) не зная HTML и CSS.

Однако, для того чтобы сделать сайт с помощью CMS потребуются услугии программиста, и дизайнера, и верстальщика. И капиталовложения.

Какие бывают cms

Бывают разлиные системы управления контентом: для интернет-магизонов, для блогов, для форумов и т.д.

Примеры cms

Примеры популярных CMS: Joomla, WordPress (для блогов), PhpBB (для форумов)
